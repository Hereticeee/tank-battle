# 坦克大战 - Tank Battle

经典 FC 坦克大战的 HTML5 复刻版，纯前端实现，无需任何依赖。

## 🎮 游戏玩法

- 控制你的坦克在战场上移动和射击
- 消灭所有敌方坦克进入下一波
- 保护你的基地（鹰标）不被摧毁
- 每波敌人数量和速度都会增加

## 🕹️ 操作说明

| 按键 | 功能 |
|------|------|
| ↑ ↓ ← → 或 WASD | 移动坦克 |
| 空格键 | 发射子弹 |
| R | 重新开始 |

## 📱 触屏支持

在移动设备上，触摸拖动来控制移动方向，自动射击。

## 🎯 游戏特性

- 经典俯视角坦克大战玩法
- 可破坏的砖墙和不可破坏的钢铁墙
- 敌方 AI 会追踪玩家位置
- 分数系统和波次递增
- 粒子爆炸特效
- 3条生命，生命耗尽游戏结束
- 移动端触屏操作支持

## 🚀 快速开始

1. 克隆仓库
```bash
git clone https://github.com/YOUR_USERNAME/tank-battle.git
```

2. 直接用浏览器打开 `index.html` 即可开始游戏

或者使用任意静态服务器：
```bash
npx serve .
# 或
python -m http.server 8000
```
